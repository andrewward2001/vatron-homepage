About Vatron

Vatron is a brand new VATSIM map client built on top of modern, cross-platform technologies. Vatron promises to be fast and lightweight, while packing some of the most advanced technologies current available. Vatron is written entirly in the language of the internet, meaning that it has one of the largest development communities making up each and every component.

Contribute to Vatron

Vatron is totally free and open source, forever. Head over to the GitHub repo to contribute.

Before you start

Make sure Vatron the latest version has successfully built on your platform.

Windows:

macOS:

It's easy to get the source code up and running to try out the newest Vatron features.

You'll need a few things first:

  • Git or git client
  • Node.js 9.2 or newer
  • npm (bundled with Node.js) or yarn (recommended)

Open up a command prompt or Terminal and navigate to a suitible parent directory (~/Desktop, for example).

Clone the Git repo and enter the directory

git clone https://github.com/andrewward2001/vatron
cd vatron

Now you'll need to install the required dependencies. If you're using npm, enter

npm install

If you opted to use yarn, use

yarn

From there, it's as easy as running Vatron:

npm start

or

yarn start
